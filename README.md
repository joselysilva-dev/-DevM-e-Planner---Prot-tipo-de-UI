# 👩‍💻 DevMãe Planner - Protótipo de UI

Este repositório apresenta o protótipo de interface (UI) para o aplicativo **DevMãe Planner**, um projeto conceitual desenvolvido durante o meu primeiro mês no curso de Análise e Desenvolvimento de Sistemas (ADS).

## 🎯 Sobre o Projeto

O **DevMãe Planner** foi idealizado para ser uma ferramenta de organização centralizada para mães que atuam ou estudam na área de tecnologia. O objetivo é auxiliar no equilíbrio entre a carreira tech, os estudos constantes e a rotina familiar.

O slogan **"Organizando seu sonho..."** reflete a missão de dar suporte a essas mulheres para que alcancem seus objetivos profissionais sem deixar de lado a organização pessoal.

## 🎨 Design e Experiência do Usuário (UI/UX)

O design foi construído com foco em:
- **Simplicidade:** Interface direta para facilitar o uso em rotinas corridas.
- **Identidade Visual:** Uso de uma paleta de cores acolhedora e moderna, buscando transmitir calma e foco.
- **Acessibilidade:** Ajustes de contraste e hierarquia visual para uma leitura clara e confortável.
